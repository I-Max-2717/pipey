# Pipey. The smart alternative

Pipey is an package manager for linux. Its smart, simple and fast.

## Introduction

More features than normal package manager`s.
Time mashines are the new standard. Download packages with various versions.

``bash
./pipey install myprog 2024-02-04
``

### How to use.
You must build the program yourself but we maked a Makefile to make it really easy!
``bash
make
``
